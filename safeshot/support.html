What SafeShot Does

SafeShot helps you review screenshots and images before sharing them. It can run local OCR, detect common sensitive information, preview redaction areas, add manual redaction regions, and export or copy a redacted image.

Supported Workflows

  • Scan an image from the clipboard.
  • Open a local image file.
  • Capture a local screen area, if macOS permission is granted.
  • Review detected sensitive regions.
  • Manually add redaction areas by dragging on the image preview.
  • Copy the redacted image to the clipboard or export it as PNG.

Common Questions

Why is the first scan slower?

The first scan after launching the app may take longer because macOS loads local OCR and image-processing components. Later scans are often faster during the same session.

Why did SafeShot miss some text?

SafeShot uses Apple Vision OCR and local rules. OCR accuracy can be affected by low resolution, blur, tiny text, unusual fonts, partial screenshots, dark backgrounds, or text embedded inside another screenshot. Please manually review the image before sharing.

Why did SafeShot mark something that is not sensitive?

SafeShot is designed to be cautious. Some numbers, names, handles, codes, or URLs may be marked even when they are not private. You can uncheck individual detections before exporting or copying.

How do I add a manual redaction area?

After loading an image, drag directly on the image preview to draw a manual redaction region. The region can be included in the final redacted image.

Does SafeShot upload my images?

No. SafeShot is designed for local processing on your Mac. It does not upload images, OCR text, detection results, or redacted output.

Permissions

  • Files: SafeShot can access files you explicitly choose through the macOS file picker.
  • Clipboard: SafeShot can read an image you choose to scan from the clipboard and can write the redacted image back to the clipboard when requested.
  • Screen Recording: If you use local area capture, macOS may ask for Screen Recording permission so the app can capture the selected screen area.

Troubleshooting

  • If clipboard scanning does not work, copy an image again and retry.
  • If local area capture does not work, check macOS System Settings > Privacy & Security > Screen Recording.
  • If detection seems incomplete, try using a higher-resolution image or manually add redaction areas.
  • If export fails, choose a different local folder where you have write permission.
  • If the app behaves unexpectedly, quit and reopen SafeShot, then try again.

Contact Support

For technical support, bug reports, or feature requests, please contact the developer through the support contact listed on the SafeShot App Store product page or the project page where this support page is hosted.

When reporting a problem, please include your macOS version, SafeShot version, the action you were trying to perform, and whether the issue happens with a sample image that does not contain private information.

简体中文

SafeShot 的用途

SafeShot 用于在分享截图或图片前进行本地隐私检查。它可以运行本地 OCR、检测常见敏感信息、预览打码区域、添加手动打码区域，并将打码后的图片复制到剪贴板或导出为 PNG。

支持的操作

  • 扫描剪贴板中的图片。
  • 打开本地图片文件。
  • 在 macOS 授权后截取本地区域。
  • 查看检测到的敏感区域。
  • 在图片预览中拖拽添加手动打码区域。
  • 复制打码后的图片，或导出为 PNG 文件。

常见问题

为什么第一次扫描比较慢？

应用启动后的第一次扫描可能需要加载 macOS 本地 OCR 和图片处理组件，因此会比后续扫描更慢。同一会话中的后续扫描通常会更快。

为什么有些文字没有识别出来？

SafeShot 使用 Apple Vision OCR 和本地规则检测。低清晰度、模糊、小字、特殊字体、局部截图、深色背景或截图里的截图，都可能影响识别结果。分享前仍建议手动检查图片。

为什么有些内容被误标为敏感？

SafeShot 的检测策略偏谨慎。一些数字、姓名、账号、代码或链接即使并非隐私信息，也可能被标记。你可以在导出或复制前取消勾选单项检测结果。

如何添加手动打码区域？

载入图片后，直接在图片预览区域拖拽即可添加手动打码区域。该区域会参与最终打码输出。

SafeShot 会上传图片吗？

不会。SafeShot 设计为在你的 Mac 本机处理图片。它不会上传原图、OCR 文本、检测结果或打码后的图片。

权限说明

  • 文件：SafeShot 只能访问你通过 macOS 文件选择器主动选择的文件。
  • 剪贴板：SafeShot 可以读取你选择扫描的剪贴板图片，并在你点击复制时写入打码后的图片。
  • 屏幕录制：如果使用本地区域截图，macOS 可能会要求授予屏幕录制权限，用于截取你选择的屏幕区域。

故障排查

  • 如果剪贴板扫描失败，请重新复制图片后再试。
  • 如果区域截图不可用，请检查 macOS 系统设置 > 隐私与安全性 > 屏幕录制。
  • 如果检测不完整，请尝试使用更高清的图片，或手动添加打码区域。
  • 如果导出失败，请选择一个你有写入权限的本地文件夹。
  • 如果应用表现异常，请退出并重新打开 SafeShot 后再试。
