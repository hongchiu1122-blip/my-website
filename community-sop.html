3. Reddit — 官方 Subreddit

🤖
Reddit Official Subreddit
Reddit 没有官方认证徽章体系，官方身份主要依赖 User Flair（用户标签）+ 品牌视觉 + 官方描述 + 官网互链来体现。 创建社区时，Reddit 官方明确要求社区名称为 3–21 字符，且创建后不可修改；账号创建权限可能受账号状态、反垃圾系统和社区行为影响，因此建议提前养号并保持正向 Karma。

🚪 步骤 1–2：前置条件

  • 前往 reddit.com/register 注册账号
  • 建议提前使用企业邮箱或统一管理账号养号，保持正常浏览、评论和互动记录
  • 保持正向 Karma，避免短期大量外链、重复宣传或跨社区刷屏
  • Reddit 未在当前官方创建社区帮助页中公开固定“30 天 + Karma”硬门槛；实际创建权限可能按账号状态动态判定
  • 确认四条件：已有 Reddit 讨论你的游戏 / 能持续运营 / 有专人负责 / 能接受公开批评

📛 步骤 3：创建 Subreddit

  • 前往 reddit.com/subreddits/create
  • 命名建议：r/GameName 或 r/StudioName
  • 名称必须为 3–21 字符，并构成社区 URL
  • ⚠️ 名称创建后不可修改，包括大小写，提交前必须确认法务、商标和品牌拼写
  • Topic 选择：Gaming；Community Type：Public

🎨 步骤 4：社区装修

  • Avatar：256×256 PNG，使用游戏 Logo
  • Banner：Desktop 1920×384；Mobile 1600×480
  • About / Description 写明官方身份、官网、Discord、Steam、客服入口
  • 设置 Welcome Message，新成员加入时自动展示规则和常用链接

🏷️ 步骤 5：Flair & 规则

  • User Flair：Developer / Community Manager / Support / Moderator
  • Post Flair：Official News / Patch Notes / Bug Report / Feedback / Guide / Meme
  • 设置社区规则：禁止人身攻击、剧透规范、Bug 反馈模板、广告限制
  • 官方人员发帖统一使用 User Flair，并在首帖中说明官方账号列表和权限边界

5. YouTube — 品牌频道

▶️
YouTube Brand Channel
YouTube 认证徽章是频道名称旁的认证勾号（可在对外中文材料中称为“灰色认证勾号 / Verification Check Mark”）。申请门槛为 100,000 订阅者，且对应 Google 账号必须开启 2-Step Verification。申请后 YouTube 会审核频道是否真实、完整、公开并保持活跃。 短期内应先完成手机号验证、频道基础装修、@handle 和权限管理。

📋 步骤 1–2：创建 Brand Channel

  • 使用企业 Google Workspace 邮箱（非个人 Gmail），前往 youtube.com/account
  • 创建 Brand Channel，避免频道资产绑定员工私人账号
  • Settings → Permissions 添加管理员：社区经理、市场负责人、备用管理员
  • Google 账号开启 2-Step Verification
  • 参考：YouTube 频道权限管理

📱 步骤 3：手机号验证与功能解锁

  • 进入 YouTube Studio
  • Settings → Channel → Feature Eligibility
  • 完成 Phone Verification 后可解锁部分基础功能，如自定义缩略图、较长视频和直播相关功能；实际可用项以 Studio 后台为准

✅ 步骤 4–5：认证徽章

  • 申请门槛：100,000 订阅者 + Google 账号 2-Step Verification
  • 频道需真实代表品牌 / 公司 / 创作者，并保持公开、完整、活跃
  • 完整性包含频道横幅、简介、头像和已发布内容
  • 更改频道名称会导致频道失去认证状态并需重新申请；更改 handle 不会移除认证徽章

📌 建议做的事

  • 创建后立即上传品牌视觉：横幅（2560×1440，安全区 1546×423）、Logo（98×98）
  • 配置频道预告片、播放列表、频道分区和官网 / 社交链接
  • 使用 @handle 作为品牌识别主路径；YouTube 新自定义 URL 已不再开放设置，旧 URL 会继续跳转到基于 handle 的新 URL
  • 避免把 YouTube 认证写成“蓝V”；对外统一称为“认证勾号”或“灰色认证勾号”

X Premium Business（原 Verified Organizations）

  • Bio 明确官方身份、官网、客服入口、Discord / Steam 链接
  • 官网 ↔ X 双向互链，提高身份可信度

💰 Premium Business 费用口径

  • Basic：美国地区公开价格 $200/月或 $2,000/年；审核通过后可获得金色勾号和方形头像，包含 Priority Support、Premium+、Hiring、Radar Basic 等功能；不含 Affiliations
  • Full Access：美国地区公开价格 $1,000/月或 $10,000/年；审核通过后可获得金色勾号，包含 Affiliations、VIP Support、Premium+、Hiring、Radar Pro 等功能
  • Affiliates：美国地区公开价格 $50/月/账号或 $600/年/账号，适用于 Full Access；Basic 不包含 Affiliations
  • Enterprise / Custom：定制商务方案，价格、广告额度和支持权益以 X 商务页面或后台报价为准，不按标准自助套餐排期

📄 申请材料 & 流程

  • 前往 x.com/settings/verified-organizations 或 X Premium Business 帮助
  • 提交：组织名称、运营国家、官网 URL、组织类型
  • 准备：营业执照 / 主体证明、域名所有权证明、申请人身份证件
  • 审核周期无固定公开 SLA，以 X 后台状态和邮件通知为准
  • 常见被拒原因：组织信息不一致、品牌不匹配、使用通用邮箱、官网不可访问

👥 步骤 4：配置 Affiliates

  • 用途：社区经理账号、地区官号、客服号、品牌子账号
  • Affiliates 属 Full Access 权益，Basic 不包含 Affiliations
  • 被邀请账号需主动接受邀请后关联生效
  • 附属账号显示母品牌小头像图标 + 认证标识

📋 步骤 1–2：注册 & 切换 Business Account

  • 使用企业域名邮箱（如 @yourstudio.com）前往 tiktok.com/signup 注册
  • Settings → Account → Switch to Business Account
  • 完善：Logo（200×200）、Bio、官网链接、官方邮箱
  • 开启 2FA / Multi-factor Authentication，绑定已验证邮箱

🏢 TikTok Business Verification

  • 适用于 TikTok Ads Manager / TikTok Business Center，用于验证企业与广告主身份
  • 准备营业执照 / 注册证明等文件，文件需清晰、完整、未过期，大小不超过 10MB
  • Business Center 企业信息提交后状态会进入 Pending，官方帮助页说明审核可能最长约 2 个工作日
  • Business Verification 不等于账号获得公开展示的蓝勾 Verified Badge

✅ Verified Badge 认证条件

  • Active：账号在过去 6 个月内登录过 TikTok
  • Authentic：代表真实个人、企业或实体，通常一个人或企业仅对应一个认证账号
  • Complete：公开账号，资料完整，包含名称、Bio、头像和至少一条视频
  • Notable：需要多篇独立权威媒体报道；新闻稿、赞助内容和付费报道不计入
  • Secure：开启多因素认证并绑定已验证邮箱

📝 申请流程与提醒

  • TikTok App 内：Profile → Menu → Settings and privacy → Account → Verification → Start（入口按地区和账号状态可能不同）
  • 提交企业 / 品牌身份证明和多篇独立媒体报道链接
  • Verified Badge 没有统一固定审核周期，以 App / 后台显示为准
  • 任何声称可付费购买 TikTok 认证徽章的服务均不可信
  • 认证后更改 username 会触发认证徽章移除，需重新申请或重新评估；申请前应锁定品牌账号名、username、官网和媒体报道口径
  • PR 布局建议提前做，按“multiple news sources / 多篇独立媒体报道”准备，不以固定篇数作为官方硬门槛
